# Research Boundaries

Singularity Agent Tools exposes research context, not personalized investment advice.

Agent responses should:

- Label outputs as research-only.
- Include the data date when available.
- Distinguish facts, model scores, and inferred interpretation.
- Mention relevant risks and missing data.
- Avoid guaranteed returns, personalized buy/sell instructions, or target prices unless the upstream platform explicitly supports those fields with compliant wording.
